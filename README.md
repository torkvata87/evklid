# Одностраничный веб-сайт для демонстрационной компании "Проектные решения"

## Описание проекта

Этот проект представляет собой одностраничный веб-сайт для демонстрационной компании "Проектные решения", предлагающей проектные решения для бизнеса. Целью создания данного сайта было отработать навыки верстки, адаптивного дизайна и оформления форм ввода.

## Функциональность

- **Главная страница:** содержит информацию о компании, предоставляемых услугах и контактные данные.
- **Форма обратной связи:** позволяет пользователям отправить сообщение компании.
- **Адаптивный дизайн:** сайт корректно отображается на устройствах с различными разрешениями экранов.

## Технологии

- **HTML:** для создания структуры страницы.
- **CSS:** для оформления и стилизации.
- 
## Особенности

- **Адаптивный дизайн:** сайт оптимизирован для отображения на мобильных устройствах, планшетах и настольных компьютерах.
- **Оформление форм:** использованы различные техники для стилизации форм ввода.
- **Кроссбраузерность:** сайт протестирован в нескольких популярных браузерах.
